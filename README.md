📄 Phishing Email Detection & Awareness System
Cyber Security Task 2 (2026)
By Future Interns

🔍 Project Overview
Phishing is one of the most common and costly cyber-attacks faced by businesses today. Attackers trick users into:
- clicking malicious links
- downloading infected files
- sharing passwords or OTPs
Most successful phishing attacks happen not because of weak systems, but because users don’t know how to identify fake emails.
This project simulates the role of a security analyst: analyzing phishing emails, identifying red flags, and creating a professional awareness report that businesses can use to train employees.

🎯 Objectives
- Analyze real phishing email samples
- Identify common phishing indicators
- Classify email risk clearly
- Explain attacks in simple, non-technical language
- Create awareness guidelines to help users avoid phishing attacks

🛠️ Tools Used
- Email Header Analysis
- Google Toolbox Header Analyzer
- MXToolbox Email Header Analyzer
- Browser Tools: For checking links/domains safely
- Documentation Tools: MS Word / PDF

📧 Sample Phishing Email
Example analyzed in this project:
Subject: ⚠️ Urgent: Fraudulent Expenses Detected
Sender: human.resources@alerting-services[.]com
Body: Claims fraudulent expenses were flagged and urges the recipient to click “View Fraudulent Expense Details.
